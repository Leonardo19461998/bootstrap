Sobre el juego

Esta aventura gráfica está inspirada en Avatar: La Leyenda de Aang, combinando elementos visuales de las cuatro naciones: Aire, Agua, Tierra y Fuego.

El juego fue desarrollado completamente en JavaScript con p5.js, aplicando Programación Orientada a Objetos (POO). Se diseñaron múltiples clases para estructurar las mecánicas y las pantallas del juego.

Para los gráficos probamos varias imágenes generadas con IA y algunas las ajustamos para que encajen con los fondos de cada nación. También armamos animaciones simples y usamos detección de colisiones con p5.js.
